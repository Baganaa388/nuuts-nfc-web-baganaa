# Mazaalai Conservation Leaderboard

NFC карт ашиглан хэрэглэгчдийн хөнгөлөлтийн **leaderboard** харуулах жижиг систем. 
Энэхүү систем нь **Говийн эзэн Мазаалай** болон Монголын байгалийг хамгаалах бодит баатруудыг хүлээн зөвшөөрч, тэдний хувь нэмрийг тэмдэглэх зорилготой.

## Гол санаа

- Хэрэглэгч бүр NFC түлхүүрийн оосортой.
- NFC → хэрэглэгчийн бүртгэлтэй ID / profile холбогдоно.
- Admin NFC уншуулсан хэрэглэгчийн дүнг нэмбэл:
  - `leaderboard` шууд шинэчлэгдэнэ.
- Бүртгэлгүй NFC байвал:
  - Register page рүү хөтлөөд хэрэглэгч шинээр үүсгэнэ.
  - Дараагийн удаа NFC уншуулбал тухайн хэрэглэгчийн profile линк рүү ордог болно.
- ESP32 эсвэл gateway скрипт:
  - UID-ыг сервер рүү автоматаар шиднэ.
  - NDEF URL writer нь серверээс profile / register URL асууж NFC карт руу бичнэ.
