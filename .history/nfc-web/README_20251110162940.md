## üé® Design System & UI Improvements

### Brand Identity
- **Primary Color**: Warm brownish mustard yellow (#D4A574) - inspired by Mongolian Soyombo yellow
- **Supporting Colors**: 
  - Deep forest green (#2D5016) for nature theme
  - Charcoal (#1A1A1A) for text
  - Soft off-white (#FAF8F3) for backgrounds
  - Earth brown accents (#8B6F47)

### Design Features
- **Mobile-First Responsive Design**: Optimized for mobile phones (360-430px), scales beautifully to tablet and desktop
- **Modern Typography**: Clear hierarchy with system fonts, optimized line heights and spacing
- **Smooth Animations**: 
  - Fade-in animations for sections on page load
  - Staggered animations for leaderboard rows
  - Button hover/tap feedback with scale and shadow transitions
  - Card elevation changes on hover
  - Subtle gradient animations in hero sections
- **Accessibility**: Respects `prefers-reduced-motion` for users who prefer minimal animations
- **Design Tokens**: Centralized color, spacing, typography, and animation variables in CSS custom properties

### Pages & Components
- **Hero Section**: Introduces the Mazaalai conservation project on the homepage
- **About Page** (`/about`): Explains the Mazaalai bear conservation mission
- **Improved Forms**: Modern input styling with focus states and smooth transitions
- **Enhanced Leaderboard**: Better visual hierarchy, rank indicators, and hover effects
- **Consistent Admin UI**: Admin interface uses the same design system for consistency

### Animation Details
- **Page Load**: Sections fade in with slight upward motion (200-400ms)
- **Buttons**: Hover scale-up with shadow, tap scale-down feedback
- **Cards**: Subtle elevation change and slight movement on hover
- **Leaderboard Rows**: Staggered fade-in animations for visual interest
- **Navigation**: Smooth transitions for all interactive elements

### Design System Location
All design tokens (colors, spacing, typography, animations) are defined in `layouts.js` at the top of the CSS, making it easy to customize:
- Colors: `:root` CSS variables starting with `--brand-`, `--color-`, `--bg-`, `--text-`
- Spacing: `--space-xs` through `--space-3xl`
- Typography: `--font-size-*` and `--line-height-*`
- Animations: `--duration-*` and `--easing-*`
